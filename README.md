## 🛠️ How to Add a New Project

**1. Create a new folder:**

```bash
mkdir your-project-name
```

**2. Add your `index.html` inside it:**

```
projects/
  your-project-name/
    index.html
```

**3. Push to GitHub:**

```bash
git add .
git commit -m "Add: your-project-name"
git push
```

**4. Deploy on Vercel:**

- Go to [vercel.com](https://vercel.com) → **Add New** → **Project**
- Import the `projects` repo
- Set **Root Directory** to `your-project-name`
- Set **Output Directory** to `.`
- Click **Deploy** ✅

**5. Update this README** with the new project and its live URL.
